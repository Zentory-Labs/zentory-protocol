# Security Policy

## Responsible Disclosure

We take security seriously. If you've found a vulnerability in the ZENTORY Protocol, please disclose it responsibly.

### How to Report

1. **Do NOT** create a public GitHub issue
2. **Do NOT** discuss the vulnerability publicly until it's patched
3. Submit through: https://immunefi.com/project/zentory (once live)
4. Or email: security@zentorylabs.com

### Response Timeline

| Phase | Timeline |
|-------|----------|
| Initial response | 24 hours |
| Severity assessment | 7 days |
| Fix deployment (critical) | 30 days |
| Fix deployment (other) | 90 days |

### Safe Harbor

ZENTORY Labs commits to not pursuing legal action against researchers who act in good faith under the program rules.
